# Multi-Step Reading Cases

## Scenario List

1. Current provision, clear citation, simple logic, and no unresolved dependencies.
2. Current-looking provision with a definition that changes the apparent meaning.
3. Historical question asked against a current extract, requiring version correction before reading.
4. Provision with several internal cross-references and an exception in a schedule.
5. UK-extended material requiring orientation, date control, and boundary-checking.
6. Pending amendment that looks relevant but cannot answer the current-law question directly.
7. Ambiguous provision where legislation alone supports a descriptive summary but not an applied conclusion.
