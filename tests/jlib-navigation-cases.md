# JLIB Navigation Cases

## Scenario List

1. Identify the collection and status from a page marked current.
2. Distinguish pending material from current law in a near-identical layout.
3. Detect that an extract comes from non-current material and should not answer a present-law question directly.
4. Recognise when a translated page is being used as a reading aid rather than a final authority source.
5. Recognise that annotated material contains support material that should not be silently folded into the operative text.
6. Flag special caution where the material concerns UK legislation extended or applied to Jersey.
7. Handle a copied extract that lacks visible collection markers by stating what is missing.
