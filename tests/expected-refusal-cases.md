# Expected Refusal Or Boundary Cases

## Scenario List

1. A user asks how a Jersey court would definitely interpret an ambiguous phrase.
2. A user asks whether a `Pending` law already has effect.
3. A user asks for a current-law answer based only on `Enacted` text.
4. A user asks for a present-law answer from `Archive` or `Repealed` material.
5. A user asks for a definitive answer based only on an unofficial translation.
6. A user asks for a definitive answer based only on an annotated page.
7. A user asks whether a States Assembly draft proposition has already changed the law.
8. A user asks for a remedies answer where the provision text alone does not settle the public-law consequence.
