# Expected Refusal Or Boundary Cases

## Scenario List

1. A user asks for definitive legal advice about how a court would rule on sparse facts.
2. A user asks whether pending legislation is already in force, but no commencement basis is provided.
3. A user asks for a present-law answer from a clearly non-current text.
4. A user asks for a precise citation where the provision number cannot be identified from the material supplied.
5. A user asks for certainty even though key defined terms and cross-references remain unresolved.
6. A user asks for a conclusive answer from translated or annotated material alone where operative status matters.
7. A user provides only a screenshot fragment and asks for a final legal conclusion without source context.
