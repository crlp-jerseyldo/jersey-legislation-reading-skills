# JLIB Research Navigation

## Purpose

This note explains how to get to the right JLIB legislation page before source-state classification begins. It is an operational aid grounded in official JLIB public guidance, not an official platform manual.

## Official JLIB Guidance Used Here

- Search guide: `https://www.jerseylaw.je/faqs/pages/search-guide.aspx`
- Current laws FAQ: `https://www.jerseylaw.je/faqs/Pages/current-laws.aspx`
- Types of legislation FAQ: `https://www.jerseylaw.je/faqs/Pages/types-of-legislation.aspx`
- Laws as enacted FAQ: `https://www.jerseylaw.je/faqs/Pages/Laws-as-enacted.aspx`

## Starting Rule

Operational recommendation from the official JLIB search guide:

- use the homepage search bar for whole-site searching
- for specific legislation research, go to the Laws search page first

That recommendation keeps legislation research anchored to the Laws area instead of mixing in judgments, publications, or other site content too early.

## Choose The Entry Path

Start from the Laws search page unless the task is genuinely whole-site.

Then route by question type:

- present law -> `Current`
- historical question from `1 January 2019` onward -> find the law in `Current`, then use point-in-time
- original text -> `As Enacted`
- future or not-yet-effective legislation -> `Pending`, and often also `As Enacted`
- older historical or non-official material -> `Non-current`
- unofficial English aid for older French law -> `Translated`
- article-level case-linking context -> `Annotated`

This routing is an operational recommendation built from JLIB's collection guidance. It should be read alongside the repo's source-state map in `docs/jlib-collections-map.md`.

## Search Moves Supported By Official JLIB Guidance

If the exact title or citation is known:

- search it in quotation marks first

If the exact title is not known:

- search topic words on the Laws page
- narrow with filters or sorting on the relevant page

The official search guide says search pages support filtering and sorting such as:

- A-Z
- oldest / newest
- content type
- date range
- subjects
- offences
- judges

Not every filter is relevant to legislation research, but content type, date range, and sorting are often useful.

## Official Search Syntax

Use only the official features the site documents:

- quotation marks for exact phrases
- `AND` to require both terms
- `OR` to allow alternatives
- `NOT` to exclude false positives
- `NEAR` for terms close together without preserving order
- `ONEAR` for proximity with order preserved
- `WORDS(...)` to treat terms as synonyms for ranking purposes
- `*` as a wildcard for word stems or uncertain spelling

The official search guide also documents `XRANK`, but for ordinary legislation research it is usually unnecessary. Treat it as an advanced feature rather than a normal first-line move.

## Current Laws Navigation

The official Current laws FAQ gives several navigation moves that matter in research:

- the Current laws page contains both primary legislation and secondary legislation
- JLIB says the page has a view toggle for primary and secondary laws
- the page date statement near the top of a law helps confirm when the current version took effect
- every current law has endnotes giving history about original making and amendments

Operational consequence:

- use the current-law page as the main hub for present-law reading
- check the displayed date statement before treating the page as the right version
- use endnotes for amendment history context, but not as a substitute for source-state checks

## Point-In-Time Retrieval

The official Current laws FAQ says point-in-time versions are available back to `1 January 2019`.

To retrieve one:

- go to the current law you need
- on desktop, open the point-in-time timeline
- on mobile, use the version dropdown
- select the version start date or search for a specific date

Each version states the period it was in effect. All dates are inclusive.

## Future Amendments

The official Current laws FAQ says amendments that have been enacted or made but are not yet commenced are not included in the current law.

Operational consequence:

- if the research question is about a recent change that may not yet be in force, do not stop at `Current`
- check `As Enacted` for the recent amending instrument as well

## Reading Aids And Context Aids

Official JLIB guidance says:

- translated laws are unofficial English aids for some French laws
- annotated laws are contextual material from the superseded revised edition

Operational consequence:

- `Translated` can help understanding, but the authoritative text remains the original French law
- `Annotated` can help locate case-linked context, but operative reading should return to `Current` or `Enacted`
