# Walkthrough: Background Default Trigger

## Scenario

A user asks for the effect of a current Jersey provision that uses singular drafting in a way that may affect whether the rule applies more broadly.

The page and provision have already been identified.

## Suggested Skill Sequence

1. `01-orient-the-text-on-jlib`
   Record the source state, officiality, date range, and provision in view.
2. `02-find-the-right-version-and-date`
   Confirm that the source matches the present-law question.
3. `03-cite-and-link-jersey-legislation-properly`
   Fix the provision-level reference and preserve the page date context.
4. `04-read-a-provision-as-logic`
   Restate the rule and identify the unresolved singular-or-plural issue.
5. `05-apply-definitions-and-cross-references`
   Confirm that no local definition, instrument-wide interpretation provision, or direct cross-reference settles the issue.
6. `06-apply-non-obvious-background-rules`
   Check the relevant default from the `Interpretation (Jersey) Law 1954` explicitly and record whether it changes the reading.
7. `07-boundary-check-legislation-alone-or-not`
   Separate the clarified legislative reading from any fact-sensitive or broader legal conclusion.

## Safe Outcome

Use `06` only because a real Jersey-wide background-rule issue remains after `05`. If `05` had already settled the point directly, the workflow would not need this extra step.
