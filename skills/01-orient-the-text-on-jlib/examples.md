# Examples

## Example 1: Current Point-In-Time

Input: A JLIB page in the current collection that says `Showing the law from 11 January 2024 to 23 November 2025`.

Use: Record `Current point-in-time`, mark it as official consolidated material, capture the page date range, and note that it may answer a historical question within that period.

## Example 2: Pending

Input: A JLIB page in `Pending Laws` showing adoption but blank later dates for sanction, registration, or commencement.

Use: Record `Pending`, mark it as not yet current, and warn against treating it as law already in force.

## Example 3: Translated

Input: A JLIB page in `Translated Laws` that states the English text is not an authoritative translation.

Use: Record `Translated`, mark it as unofficial, and note that the French original remains the authoritative text.

## Example 4: Annotated

Input: A JLIB page in `Annotated Laws` with article-level references to reported cases.

Use: Record `Annotated`, note that it is a contextual aid rather than a current-law substitute, and direct the reader back to the current or enacted text for the operative provision.
