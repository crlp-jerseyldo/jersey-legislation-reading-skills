# Tests

## Prompt Set

1. You are shown a JLIB page from `Pending Laws`. Produce only the orientation block.
2. A page is in the current collection but concerns UK legislation extended to Jersey. What special caution must be recorded?
3. A user pastes only provision text and omits the page header and breadcrumb. What must the orientation output say is missing?
4. A page states `This law was repealed and is no longer in force`. How should the source state be recorded?
5. A translated page states that the English text is not an authoritative translation. How should officiality be recorded?
6. A reader supplies an annotated page and asks for a current-law answer. What should the orientation summary warn about?
7. A page shows an official consolidated statement and a date range ending in `Current`. What fields must be captured before substantive reading?
