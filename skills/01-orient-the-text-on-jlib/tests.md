# Tests

## Prompt Set

1. You are shown a JLIB extract with a visible "non-current" marker. Produce only an orientation summary.
2. A user pastes text from a page but omits the collection and date context. What should be identified before interpretation?
3. A page appears to be current legislation, but also includes annotation. What caution should be stated?
4. A user asks what Article 3 means, but only supplies a screenshot header and no provision text. What is the correct first move?
5. A JLIB item appears to be UK legislation extended to Jersey. What special caution should appear in the output?
6. The status marker is unclear from the extract. How should uncertainty be handled?
