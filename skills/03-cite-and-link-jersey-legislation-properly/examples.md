# Examples

## Example 1: Current Official Jersey Legislation

Input: A current JLIB page showing `Article 7`.

Use: Cite the instrument title plus `Article 7`, record `Current` and the page date range, and include the direct JLIB link.

## Example 2: Current Point-In-Time

Input: A point-in-time page effective from `11 January 2024` to `23 November 2025`.

Use: Cite the provision, record `Current point-in-time`, and preserve that date range in the citation note.

## Example 3: Enacted

Input: A user asks for the provision as originally made.

Use: Cite the provision as `Enacted` and note that the source is original text rather than updated current law.

## Example 4: Translated

Input: A translated page for an older French law.

Use: Cite the provision and mark clearly that the English translation is unofficial and used as a reading aid.
