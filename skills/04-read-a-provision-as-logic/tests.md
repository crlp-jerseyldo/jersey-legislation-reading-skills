# Tests

## Prompt Set

1. A provision contains a main rule and an express exception. How should the output be structured?
2. A provision has three alternative limbs in subparagraphs. What must not be lost in simplification?
3. The text uses a defined term that has not yet been checked. Should the logic map treat it as settled?
4. A provision cross-refers to another article that may qualify its effect. What should the output include?
5. A user asks for a one-line answer, but the provision is condition-heavy. How should false simplicity be avoided?
6. The provision appears incomplete because only one paragraph is supplied. What uncertainty should be signalled?
