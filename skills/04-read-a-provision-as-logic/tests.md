# Tests

## Prompt Set

1. A provision contains a main rule and an exception in paragraph `(3)`. How should the structure be shown?
2. A provision contains nested `and` and `or` logic. What must the reading output preserve?
3. A term is followed by `means`. What cue should the logic map record?
4. A term is followed by `includes`. What caution should the logic map record?
5. A duty is drafted in the singular and present tense. How should false narrowing be avoided?
6. Only one limb of a multi-part provision is supplied. What special caution should appear?
