# Tests

## Prompt Set

1. A user gives the exact title of a Jersey Law and asks for the current text. What JLIB entrypoint and search move should be used first?
2. A user asks generally about a Jersey legislation topic but has no instrument title. Why is the Laws search page a better starting point than homepage search?
3. A user asks what a law said in 2020 but only gives the law title. What page should be found first, and how should the historical version then be reached?
4. A user asks whether a recently made amendment is not yet in force. Why might `As Enacted` need to be checked even if the current law page is found?
5. A broad topic search is noisy. What official JLIB narrowing tools should be preferred before inventing a new search method?
6. A researcher is on the Current laws page and needs to separate primary from secondary legislation. What official page feature should be used?
7. A query needs exact words together in the same order. Which official search feature should be used?
8. A query needs alternatives or exclusions. Which official Boolean operators are available?
9. A query needs terms near each other. Which official proximity operators are available?
10. A query needs synonym-style matching with ranking benefits. Which official operator is available?
11. A query needs a word stem or uncertain spelling. Which official wildcard is available?
12. A user wants an English aid for an older French law. Which collection may help, and what limitation must be stated?
13. A user wants case-linked context for a law. Which collection may help, and why is that not the end of the workflow?
