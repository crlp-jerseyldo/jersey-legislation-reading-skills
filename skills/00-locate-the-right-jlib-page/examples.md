# Examples

## Example 1: Exact Title Search

Input: A user asks for the current text of `Road Traffic (Jersey) Law 1956` and no link is supplied.

Use: Start from Laws search, search the exact title in quotation marks, route to `Current`, and then hand off to orientation.

## Example 2: Broad Topic Search

Input: A user asks for Jersey legislation on tourism accommodation standards.

Use: Start from Laws search rather than whole-site search, search topic terms, and narrow with available filters or sorts before choosing a candidate current law.

## Example 3: Historical Version

Input: A user asks what a law said in 2021 but does not give a point-in-time link.

Use: Find the law in `Current`, open the point-in-time feature, select the relevant historical version, and then hand off to orientation and date control.

## Example 4: Future Amendment

Input: A user asks whether a recent amendment has been made but not yet brought into force.

Use: Check `Current`, then search `As Enacted` for recent amending legislation not yet reflected in the current text.

## Example 5: Exclusion Search

Input: A search for `housing` is returning material about social security rather than tenancy rules.

Use: Keep the search on the Laws page and refine with official operators such as `NOT` or with filters and sorting.

## Example 6: French Law Reading Aid

Input: A user wants help locating an English aid for an older French law.

Use: Route to `Translated`, mark it as a reading aid only, and preserve a return path to the authoritative French text.

## Example 7: Annotation-Led Context Search

Input: A user wants to find whether a law has article-level links to reported cases.

Use: Route to `Annotated` for context only, then move back to the operative current or enacted text for substantive reading.
