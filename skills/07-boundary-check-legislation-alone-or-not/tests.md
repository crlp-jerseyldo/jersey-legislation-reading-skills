# Tests

## Prompt Set

1. A user asks for a plain summary of what a current article says. Is legislation alone enough?
2. A user asks how a court would apply an ambiguous phrase. What additional material may be needed?
3. A provision depends on undefined facts outside the text. What should the skill say?
4. A user wants a definitive legal outcome from a brief extract. How should false certainty be resisted?
5. The text is clear descriptively but not as to practical application. How should the output separate those layers?
6. A cross-reference chain remains unresolved. Can legislation alone yet support a firm answer?
7. The material is pending legislation. Can it answer a present-law question by itself?
