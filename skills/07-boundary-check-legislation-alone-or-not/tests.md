# Tests

## Prompt Set

1. A user asks for a plain summary of a current official Article. Can legislation alone support that?
2. A user asks how a court would interpret an ambiguous phrase. What stop rule is triggered?
3. A user asks whether a draft proposition has changed the law already. What source-category problem must be identified?
4. A user wants a definitive answer based only on a translated law. What should the output say?
5. A public-law duty appears clear on the page, but the user asks about remedies for breach. What further material may be needed?
6. A question turns on the user's detailed facts rather than the bare text. How should the safe answer posture change?
7. A question depends on customary law or common law. What must be said about Jerseylaw.je limits?
